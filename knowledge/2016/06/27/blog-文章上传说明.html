搭建个人博客

文章上传操作说明

  1. 使用atom写文章，注意 date 不能超过今天。
  2. 将文章保存在文件夹 posts 内
  3. 在 iTerm 中运行命令行
  • cd myblog #打开blog文件夹
  • jekyll build #生成jekyll文件
  • jekyll serve #本地生成博客

新开一个iTerm窗口,运行如下命令行

  • cd myblog/site #输入命令行进入blog文件夹中需要上传的文件夹_site 中
  • git status #查看有哪些新增文档
  • git add . #添加所有改变到临时存储
  • git commit -m -a #推送所有改变到仓库
  • git push origin master #推送所有改变到github
